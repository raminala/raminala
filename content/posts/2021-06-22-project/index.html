This post consists of a bit of data clean up and exploratory analysis of a dataset available in University of californium, Irvine’s website. The dataset includes 17 variables and 9471 observations that represent some parameters related to air quality. This dataset is selected to perform a basic fitting.

This dataset could be find in the below link:

https://archive.ics.uci.edu/ml/datasets/Air+Quality
